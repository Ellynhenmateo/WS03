# WS03 Projects

This repository contains two separate school projects.

## Jobseeker App

A PHP MVC job listing application with authentication, job posting, search, flash messages, and a dedicated Job Match Checker page for ternary operator usage.

Main entry point:

- `public/index.php`

## Blog Project

A standalone PHP blog page with its own hero image and layout. It is kept separate from the Jobseeker app so the two projects do not mix their logic.

Files:

- `public/blog/lab1.php`
- `public/blog/hero.jpg`

Local URL:

- `/blog/lab1.php`
